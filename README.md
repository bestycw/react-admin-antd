# CoffeeAdmin - 轻量级后台管理系统

CoffeeAdmin 是一个基于最新技术栈的轻量级后台管理系统模板，专注于提供快速、简洁的开发体验。

## 技术栈

### 核心框架
- React 18
- TypeScript 5.x
- Vite 5.x (构建工具)

### 状态管理
- MobX 6.x
- MobX React Lite 4.x (React 绑定)

### UI 框架
- Ant Design 5.x
- TailwindCSS 3.x (原子化 CSS)

### 路由
- React Router 6.x (声明式路由)

### 开发工具
- ESLint (代码检查)
- Prettier (代码格式化)
- TypeScript (类型检查)

## 特性

### 1. 轻量级设计
- 最小化依赖
- 快速启动和构建
- 模块化设计

### 2. 开发体验
- 完整的 TypeScript 支持
- 路径别名支持
- 热更新
- 自动导入组件

### 3. 功能特性
- 基于角色的权限控制
- 可配置的导航菜单
- 主题定制
- 响应式设计
- 多语言支持
- 系统配置持久化
